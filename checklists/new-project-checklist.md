# New Project Checklist

- [ ] Create app repository from `templates/app-repo`.
- [ ] Create deployment repository from `templates/deployment-repo`.
- [ ] Create infrastructure repository from `templates/infrastructure-repo`.
- [ ] Create docs repository from `templates/docs-repo`.
- [ ] Decide VM/Docker Compose, Kubernetes, or staged migration path.
- [ ] Configure Gitea registry and CI secrets.
- [ ] Register DefectDojo product and engagement.
- [ ] Configure monitoring ownership and alert routes.
- [ ] Record ADRs for repo split, CI/CD boundaries, registry, DefectDojo, runtime, severity gates, and monitoring.
