# Production Readiness Checklist

- [ ] Production deploy requires manual approval.
- [ ] Deployment uses immutable image tag.
- [ ] Rollback procedure is tested.
- [ ] Staging health check passes.
- [ ] DAST ran against staging.
- [ ] Monitoring alerts exist for app down, 5xx rate, restarts, and disk pressure.
- [ ] Backup and restore ownership is documented.
- [ ] DNS and TLS renewal monitoring is documented.
- [ ] Incident response runbook exists.
- [ ] Access model is reviewed.
