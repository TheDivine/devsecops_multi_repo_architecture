# AWS Baseline Module

## Purpose

Optional starter guidance for clients using AWS.

## Baseline Topics

- account separation
- IAM least privilege
- VPC and subnet segmentation
- security groups
- DNS and ACM certificate ownership
- CloudTrail and logging
- encrypted backups
- ECR or private registry integration
- Terraform/OpenTofu remote state

This module is intentionally provider-light. Real account IDs, ARNs, and credentials belong in private infrastructure repositories.
