# Repository Access Model

## App Repository

Access:
- developers
- tech leads
- platform engineers
- security engineers

## Infrastructure Repository

Access:
- platform engineers
- senior DevOps/SRE
- restricted security/admin users

Developers usually do not need access.

## Deployment Repository

Access:
- platform engineers
- release managers
- senior developers if approved
- security engineers read-only or review access

## Documentation Repository

Access:
- broad internal read access
- write access for platform, engineering leads, and documentation owners

## Platform Repository

Access:
- platform team
- DevSecOps engineers
- senior maintainers
