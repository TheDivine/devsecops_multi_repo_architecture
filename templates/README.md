# Templates

Starter repository templates for real client projects.

- `app-repo/`: application source, CI, scans, SBOM, image build and push
- `deployment-repo/`: Docker Compose and Kubernetes deployment state
- `infrastructure-repo/`: Terraform/OpenTofu, Ansible, VM/cloud planning
- `docs-repo/`: architecture, ADRs, runbooks, access, inventory, and security model
- `platform-repo/`: optional internal platform product repo starter
- `security-repo/`: optional separate security baseline repo
- `monitoring-repo/`: optional separate observability repo

Copy templates into private implementation repositories and replace placeholders there.
