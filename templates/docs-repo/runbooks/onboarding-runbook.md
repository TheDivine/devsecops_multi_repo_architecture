# Onboarding Runbook

## Steps

1. Create app, deployment, infrastructure, platform linkage, and docs repositories.
2. Assign owners and access groups.
3. Create Gitea registry project.
4. Configure CI secrets in the app repo.
5. Configure deployment environment secrets in the deployment repo.
6. Register product and engagement in DefectDojo.
7. Configure monitoring and alert ownership.
8. Record ADRs for runtime model, severity gates, and deployment approvals.
